# X-encrypt 
![X-encrypt preview](X-encrypt.jpg)

## introduction
Code Encrypter encrypt/decrypt bash file and encrypt python in Marshal format

## Instalations
```
$ pkg update && pkg upgrade -y
$ pkg install git
$ git clone https://github.com/Whomrx666/X-encrypt.git
$ cd X-encrypt
$ bash install.sh
$ python X-encrypt.py
```

## Instructions
- **First**: Install tools according to the instructions above
- **Second**: Select one of the options according to your needs and what script you want to encrypt 
- **Third**: Enter the storage where you saved the script and enter the Script name and click enter
- **Last**: For Bash, you can move it to a folder in your memory and for Python it will automatically be saved on the SD card with the following format _enc

## For example, if the script is on the SD card
```
cd /sdcard/your script name.py or sh
```
